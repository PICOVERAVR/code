ABI:
	parameters passed in R1 -> R4
	result returned in R1 (-> R4?)
	
	int func(arg1, arg2, arg3, arg4);
		arg1 -> R1
		arg2 -> R2
		arg3 -> R3
		arg4 -> R4
		return -> R1
	
	R1 - R15: caller saved, usable as temp registers
	R16 - R31: callee saved, save before touching
